### 定义问题

糟糕的性能会让您的站点损失用户数、转化率和口碑

这里的「快」有两种，一种是「真的快」一种是「觉得快」

「真的快」：可以客观衡量的指标，像网页访问时间、交互响应时间、跳转页面时间

「觉得快」：用户主观感知的性能，通过视觉引导等手段转移用户对等待时间的关注 做好这两方面都能提升用户对网站的性能评价
